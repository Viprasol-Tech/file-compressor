# File Compressor

Compress files and directories efficiently

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/file-compressor
cd file-compressor
pip install -e .
```

## Usage

### Python

```python
from file_compressor import FileCompressor

result = FileCompressor.process("data")
print(result)
```

### CLI

```bash
python -m file_compressor "your input here"
```
